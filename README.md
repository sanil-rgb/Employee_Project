# Employee_Project
An automated data analysis project using Python and Pandas. It processes employee records from Employees.csv to calculate average payment tiers, analyze distribution across cities, and filter high-performing staff into a new report. A professional, reproducible workflow.
# Employee Data Analysis Project

## Objective
The goal of this project is to analyze employee data to understand payment trends and employee distribution across different cities using Python and the Pandas library.

## Project Files
- `Employees.csv`: The raw dataset containing employee information.
- `main.py`: The Python script that performs data cleaning, calculates metrics, and filters high-performing employees.
- `filtered_employees.csv`: The output file generated by the script.

## Prerequisites
- Python 3.x
- Pandas library (`pip install pandas`)

## How to Run
1. Ensure `Employees.csv` and `main.py` are in the same directory.
2. Open your terminal in the project folder.
3. Run the following command:
```bash
   python main.py
